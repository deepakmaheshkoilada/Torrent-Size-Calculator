This project is created to calculate the total file size of torrent(s).

To view the total size of torrents in a directory:

python .\torrent-file-size-calculator.py <path-to-torrent-file(s)>

example : 

>python .\torrent-file-size-calculator.py "C:\Users\eren\Downloads\Torrents"

To view the size of each torrent file and then the total size:

python .\torrent-file-size-calculator.py <path-to-torrent-file(s)> -v

example : 

>python .\torrent-file-size-calculator.py "C:\Users\eren\Downloads\Torrents" -v
